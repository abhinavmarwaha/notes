# Google Fu

### Techniques

* social media -> @twitter
* price -> $400
* hashtags -> #throwbackthursday
* Exclude words -> jaguar speed -car
* exact match -> "tallest building".
* range of numbers -> camera $50..$100.
* Combine searches -> marathon OR race.
